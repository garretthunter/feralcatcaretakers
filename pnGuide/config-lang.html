Languages

** IMPORTANT NOTE: THIS MODULE IS INTENDED FOR LANGUAGE TRANSLATORS; IT IS NOT DOCUMENTED IN THE GUIDE. HOWEVER, PLEASE GO THROUGH Languages -> Languages Configuration TO SET THOSE OPTIONS. **

Languages -> Languages Configuration

  • Select the language for your site: This is the "master switch" that configures your site's default language. Entry example: "English"

  • Activate ML: This is a "master switch" for enabling/disabling Multi-Lingual (ML) support on your site to support multiple languages. If your site is single-language only, you can safely set this to "No". Entry example: "Yes"

  • Activate User Flags: Do you wish to display language flag graphics to enable users to more easily select their language? The flag graphics are stored, by default, in images/flags. Entry example: "Yes" (recommended to be set to "Yes" if Activate ML (above) is also set to "Yes")
