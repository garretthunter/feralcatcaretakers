User Permissions

In some special cases you may wish to grant an individual user special privileges. This can be accomplished by using User Permissions. An example is when a specific user has agreed to moderate or manage an area of the site. To that individual user you grant the same access level as an Administrator, but limit that Administrative access to a specific area of the site. User Permissions override Group Permissions. User Permissions should be used only in special cases where the existing Group Permissions are insufficient.
